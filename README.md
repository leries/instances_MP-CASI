# instances_MP-CASI
Please find attached the benchmark instances used in our paper "Optimizing assortment and inventory on shelves when products perish". The csv files have the following structure:

* Each row provides the parameters for one item, and there are 50 items per instance, i.e., 50 rows
* Column 1: Expected value of the minimum demand
* Column 2: Standard deviation of the expected minimum demand
* Column 3: Space elasticity
* Column 4: Item profit
* Column 5: Unit cost
* Column 6: Salvage value
* Column 7: Stockout cost
* Column 9: Product with
* Column 12: Max. number of facings
* Column 14: Stock per facing
